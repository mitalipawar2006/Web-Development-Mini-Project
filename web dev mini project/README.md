# Virtual Specs Try-On Website 👓

A simple and beginner-friendly frontend web development project created using HTML, CSS, and XML.  
This project represents a virtual eyewear showcase platform where users can explore different glasses frames, view face-shape recommendations, and understand the concept of virtual try-on technology.

---

## 📌 Project Overview

The Virtual Specs Try-On Website is designed to provide an attractive and user-friendly interface for showcasing eyewear products online. The website demonstrates how virtual try-on systems help users choose suitable glasses before purchasing.

This project focuses on:
- Frontend web design
- Responsive layouts
- XML-based product data storage
- Modern UI using HTML and CSS

---

## 🚀 Features

- 🏠 Modern Home Page
- 👓 Product Showcase Section
- 😊 Face Shape Guide
- 📸 Virtual Try-On Demo Section

## 👓 Product Showcase

The website includes different eyeglasses frames with:
- Product images
- Frame names
- Prices in Indian Rupees (₹)

All product details are stored inside the XML file.

---

## 😊 Face Shape Guide

The project also provides recommendations for:
- Round Face
- Oval Face
- Square Face

to help users choose suitable eyewear styles.
